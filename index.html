Navigation project (October 2017 /January 2018 )

The Idea is to create a GPS for cyclists, the project is split in 2 (Hardware part and mobile Part ) the hardware part is based on Arduino and the communication between hardware and mobile is via Bluetooth.
The user can use the mobile app to connect to the Arduino device but also for setting the destination, saving the favorite route/places and so on.
When the user presses the button "start navigation”, the app sends the route instruction to Arduino and Arduino uses some LEDs which will alert the user when he or she has to turn right or left.
